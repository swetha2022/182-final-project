# 182-final-project

This repository contains the code for all of our final experiments. 

To download ImageNet, run the following: 
```
wget https://image-net.org/data/ILSVRC/2012/ILSVRC2012_img_train.tar --no-check-certificate

wget https://image-net.org/data/ILSVRC/2012/ILSVRC2012_img_val.tar --no-check-certificate
```

Once your dataset is downloaded, run the [official PyTorch script](https://github.com/pytorch/examples/blob/main/imagenet/extract_ILSVRC.sh) to set up your dataset.
